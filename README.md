# Smart Irrigation System (IoT Based)

##  Project Description
This is an advanced **Smart Irrigation System** built using the **ESP32** microcontroller. The project aims to optimize water usage by monitoring soil moisture levels in real-time and automating the irrigation process. Data is synced to a cloud database, allowing users to monitor and control the system remotely via a mobile application.

## Key Features
Dual Operation Modes:
    ***Auto Mode:** The system automatically triggers the water pump based on sensor thresholds (Smart Logic).
    * **Manual Mode:** Remote control of the pump (ON/OFF) via the mobile app.
* **IoT Connectivity:** Real-time data streaming to a **Database**.
* **Remote Monitoring:** View live sensor readings and system status on a customized dashboard.
* **Hardware Simulation:** Full circuit design and testing using **Proteus**.

## Hardware Components
* **Microcontroller:** ESP32 (Wi-Fi Module).
* **Sensor:** Soil Moisture Sensor (Capacitive/Resistive).
* **Actuators:** 5V/12V Relay Module & Water Pump.
* **Power Supply:** Suitable DC power source.

## Software & Tech Stack
* **Programming Language:** C++ (Arduino IDE).
* **Simulation:** Proteus.
* **IoT Platform:** Mobile App for remote monitoring and control.
* **Backend:** Database for logging moisture levels and pump status.
